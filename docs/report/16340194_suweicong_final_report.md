#### 课程学习自我总结

我在项目开发小组中担任前端和后端工程师的工作，主要工作为：

+ 前端：
    + 按钮响应函数的设计与实现
    + 与后端的交互
+ 后端：
    + 云函数的设计与实现
    + 后端与数据库的交互
+ 文档相当工作：
    + API文档

#### 最得意/或有价值/或有苦劳的工作清单：
    + 最得意工作： 在前端与后端交互中，通过云函数的回调函数来执行下个云函数，保证数据的正确
    + 有价值工作： 使用云函数来操作数据库
    + 有苦劳工作： 云函数的执行不在本地，调试起来会比较麻烦
